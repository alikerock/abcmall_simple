# Simple ABCMALL
PHP, MYSQL 심플 쇼핑몰


## 🖥️ 프로젝트 소개
상품관리, 쿠폰관리, 회원관리
<br>

### ⚙️ 개발 환경
- `PHP 8.0`
- `MYSQL`

## 📌 주요 기능
#### 로그인 
- DB값 검증
- ID찾기, PW찾기
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성
#### 회원가입
- ID 중복 체크

#### 상품 등록
- 분류 등록
- 썸네일 등록
- 추가 이미지 등록
- 썸머노트 
- 옵션 추가

#### 메인 페이지 
- 상품 출력
- 회원가입

#### 장바구니
- 장바구니 담기
- 수정
- 삭제
